# TicTacToe
Ein Java-basiertes TicTacToe Spiel mit zwei Spielmodi.

## Funktionen
- Zwei-Spieler Modus (lokal)
- Einzelspieler Modus gegen den Computer
- Eingabevalidierung
- Gewinner- und Unentschieden-Erkennung

## Technologien
- Java
- Objektorientierte Programmierung

## Spielanleitung
- Zeilen und Spalten werden von 1 bis 3 eingegeben
- Spieler 1 ist X, Spieler 2 / Computer ist O
- Wähle beim Start ob du gegen einen Freund oder den Computer spielen möchtest
